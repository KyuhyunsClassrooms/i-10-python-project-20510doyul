# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제:업무 우선순위 정하고리하는 프로그램

예) 기분과 시간에 따른 활동 추천기

---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

예) 사용자가 현재 기분과 사용할 수 있는 시간을 입력하면, 조건에 맞는 활동을 추천해 주고 싶다.

내 설명:사용자가 엄무 이름, 중요도, 마감일까지 남은 기간을 입력하면, 조건에 맞게 업무를 정리하고 우선순위를 정해주고 싶다.


---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- 업무 이름
- 중요도
- 마감일까지 남은 기간

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 우선순위 점수 계산 알고리즘: (중요도 * 10) - 남은 일수 등의 공식을 사용해 점수가 높을수록 먼저 해야 하는 일로 계산한다.
- 수를 기준으로 2차원 리스트를 정렬하거나 정렬된 형태로 출력한다.
- 

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 우선순위가 가장 높은 업무부터 순서대로 번호를 매겨 화면에 출력한다.
-
